```lisp
(A 0 n)
(* 2 n)
```

故`(define (f n) (A 0 n))`计算的是2n





```lisp
(A 1 n)
(A 0 (A 1 (- n 1)))
(A 0 (A 0 (A 1 (- (- n 1) 1))))
(A 0 (A 0 (A 0 (A 1 (- (- (- n 1) 1) 1)))))
...
```

而`(A 0 (...))` == `(* 2 (...))`

故`(define (g n) (A 1 n))`计算的是n^2



```lisp
(A 2 n)
(A 1 (A 2 (- n 1)))
(A 1 (A 1 (A 2 (- (- n 1) 1))))
(A 1 (A 1 (A 1 (A 2 (- (- (- n 1) 1) 1)))))
```

而`(A 1 (...))`== `(...)*(...)`

故`(define (h n) (A 2 n))`计算的是n^2n
